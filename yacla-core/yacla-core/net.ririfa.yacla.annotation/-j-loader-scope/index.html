JLoaderScope

@FunctionalInterface
interface JLoaderScope

Java-compatible DSL scope interface for defining configuration loaders within a shared default context using net.ririfa.yacla.Yacla.withDefaults.

This interface allows Java developers to define one or more configuration loaders that share a common net.ririfa.yacla.LoaderSettings, such as a parser, logger, or auto-update flag.

Kotlin users should prefer net.ririfa.yacla.LoaderScope which provides a more idiomatic DSL.

Example usage (Java):

LoaderSettings settings = new LoaderSettings(new YamlParser(), new ConsoleLogger(), true);

MainConfig config = Yacla.withDefaults(settings, scope -> {
    ConfigLoader<MainConfig> loader = scope.loader(MainConfig.class, b -> {
        b.fromResource("/defaults/config.yml");
        b.toFile(Paths.get("config.yml"));
    });
    return loader.getConfig();
});
