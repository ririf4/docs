FieldConfigScope

class FieldConfigScope<V>

Provides a DSL-style builder to configure fields in a configuration schema.

This class allows defining field-specific properties, validation logic, and custom behavior for configuration loading and parsing. It supports defining default values, marking fields as required, setting value ranges, and adding custom deserialization logic.
