class DefaultConfigLoaderBuilder<T : Any>(clazz: Class<T>) : ConfigLoaderBuilder<T>

Default implementation of ConfigLoaderBuilder.

This builder creates a ConfigLoader for the given configuration class, allowing fluent configuration of the input resource, target file, parser, logger, and update behavior.

Typically used internally via net.ririfa.yacla.Yacla.loader, rather than instantiated directly.
