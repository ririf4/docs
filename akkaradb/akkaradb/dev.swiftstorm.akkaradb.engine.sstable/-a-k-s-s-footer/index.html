AKSSFooter

object AKSSFooter

AKSS footer (32 bytes, Little-Endian), unified definition for reader & writer.

Layout: 0..3 : magic 'AKSS' (u32) 4 : ver (u8) = 1 5..7 : pad (u24) = 0 8..15 : indexOff (u64) 16..23 : bloomOff (u64) -- 0 if none 24..27 : entries (u32) 28..31 : crc32c (u32) -- CRC32C over [0 .. fileSize-4), 0 = omitted
