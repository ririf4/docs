PackedTable

class PackedTable<T : Any, ID : Any>(val db: AkkaraDB, kClass: KClass<T>, idClass: KClass<ID>) : Closeable

A class representing a table abstraction that packs entities for storage and retrieval in a database. Provides CRUD operations, query execution, and entity ID management.

Parameters

T

The type of the entity stored in the table. Must be a non-nullable type.

ID

The type of the identifier used for entities in the table. Must be a non-nullable type.

db

The database instance to which this table is bound.

kClass

The KClass of the entity type T, used for reflection and metadata purposes.

idClass

The KClass of the identifier type ID, used for validation and serialization purposes.
