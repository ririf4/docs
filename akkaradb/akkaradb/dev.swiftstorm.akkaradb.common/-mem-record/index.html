data class MemRecord(val key: ByteBufferL, val value: ByteBufferL, val seq: Long, val flags: Byte, val keyHash: Int, val approxSizeBytes: Int)

In-memory KV entry for MemTable and read-side indexes. Decoupled from on-disk TLV encoding (see format.api.RecordView).

Keep this tiny & hot: it sits on the put/get hot path.
