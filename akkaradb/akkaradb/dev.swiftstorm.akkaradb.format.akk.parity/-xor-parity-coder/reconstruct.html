reconstruct

open override fun reconstruct(lostDataIdx: IntArray, lostParityIdx: IntArray, data: Array<ByteBufferL?>, parity: Array<ByteBufferL?>, outData: Array<ByteBufferL>, outParity: Array<ByteBufferL>): Int

Reconstruct lost blocks (data and/or parity) up to parityCount erasures.

Return

total reconstructed blocks

All destination buffers must have ≥ blockSize remaining.

Parameters

lostDataIdx

indices (0..k-1) of lost data blocks

lostParityIdx

indices (0..m-1) of lost parity blocks

data

array of size k; present entries non-null; lost entries null

parity

array of size m; present entries non-null; lost entries null

outData

destination buffers for lost data (size == lostDataIdx.size)

outParity

destination buffers for lost parity (size == lostParityIdx.size)

Throws

IllegalArgumentException

if erasures exceed parityCount or sizes/positions are inconsistent.
