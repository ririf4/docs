XorParityCoder

class XorParityCoder(val blockSize: Int = BLOCK_SIZE) : ParityCoder

XOR parity coder (m = 1).

  • No allocations on the hot path

  • Absolute-index I/O via LE.* on raw ByteBuffer

  • 64-bit wide XOR + tail-bytes
